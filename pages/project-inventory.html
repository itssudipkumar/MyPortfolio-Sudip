Project Overview

Designed and implemented a comprehensive Inventory Management System using SQL and Oracle Database. The system was built to handle real-world stock management scenarios including product tracking, automated reporting, and stock level alerts.

What I Built

  • Designed the full relational database schema with normalised tables for products, categories, suppliers, and stock levels.
  • Created complex SQL queries for product tracking, stock reporting, and inventory analytics.
  • Implemented stock level automation — automatic alerts when products drop below reorder threshold.
  • Built stored procedures and triggers to maintain data integrity across all transactions.
  • Optimised slow queries using indexes, reducing query execution time significantly.

Key Skills Demonstrated

This project demonstrated my ability to design practical database systems, write efficient SQL, and think about data integrity at scale. Oracle Database's PL/SQL environment gave me hands-on experience with enterprise-grade database tools.

Tech Stack

Database: Oracle Database · Language: SQL, PL/SQL · Tools: Oracle SQL Developer, SQL*Plus
